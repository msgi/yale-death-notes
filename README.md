# 耶鲁大学死亡哲学课程笔记

## 项目简介

本仓库包含耶鲁大学开放课程《哲学：死亡》的学习笔记。这些笔记涵盖了从二元论与物理主义的讨论，到灵魂存在的论证，再到个人同一性、死亡本质以及自杀伦理等深刻哲学议题。

## 内容概述

笔记按照课程章节组织，主要讨论以下主题：

- **哲学基础**：二元论与物理主义的对比
- **灵魂存在论证**：从多个角度探讨灵魂是否存在
- **柏拉图哲学**：灵魂不朽的论证
- **个人同一性**：跨时空同一性、灵魂理论、身体理论与人格理论
- **死亡本质**：死亡的定义、相信自己会死、独自死去
- **死亡的价值**：剥夺理论、不朽的可能性与生命价值
- **面对死亡**：死亡的恐惧、如何在必然死亡的前提下生活
- **自杀伦理**：自杀的合理性、不确定性下的决策、自杀的道德性

## 笔记特点

- 结构化的知识点梳理
- 关键概念的明确标记
- 论证过程的逻辑分析
- 观点的对比与评价（使用✅和❌标记）

## 使用说明

1. 按照编号顺序阅读笔记，以获得完整的课程内容理解
2. 重点关注标记为✅的核心概念和正确论证
3. 参考标记为❌的错误论证，理解哲学思考中的逻辑陷阱

## 关于课程

《哲学：死亡》是耶鲁大学的一门经典开放课程，由谢利·卡根教授主讲。课程通过严谨的哲学论证，引导学生思考关于死亡、生命意义以及人类存在本质的深刻问题。
